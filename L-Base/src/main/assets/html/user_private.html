一、引言
北京方寸悦动科技有限公司（简称“方寸悦动”或“我们”）我们非常重视您的隐私保护和个人信息保护。我们为了切实保护方寸悦动用户隐私权和个人信息，优化用户体验，方寸悦动（或称“我们”）根据《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《常见类型移动互联网应用程序必要个人信息范围规定》等法律法规，并参考《信息安全技术个人信息安全规范》等国家标准，制定本《隐私协议政策》。
《隐私政策》将向您概括说明我们收集和使用您个人信息的概况、告知您如何管理个人信息并行使相应权利、我们对未成年人个人信息的保护政策、《隐私政策》的适用范围及我们的联系方式。《隐私政策》正文中向您详细阐述关于每个功能可能收集的个人信息范围、收集目的、收集方式、拒绝提供个人信息可能对您产生的影响，我们如何使用cookie和同类技术，我们如何共享、转让、公开披露您的个人信息，我们如何保存及保护您的个人信息，您行使个人信息权利的具体方法，本《隐私政策》如何更新以及适用法律等内容。本《隐私政策》与您所使用的方寸悦动服务以及该服务所包括的各种业务功能息息相关，希望您在使用我们的产品与/或服务前，仔细阅读并确认您已经充分理解本条款所写明的内容，并让您可以按照本《隐私政策》的指引做出您认为适当的选择。如您想进一步了解，请查阅正文详述或通过《隐私政策》公示的联系方式与我们取得联系。

二、我们如何收集、使用您的信息以及您可以如何管理您的信息
在您使用我们的产品/服务时，我们需要/可能需要收集和使用的您的个人信息包括如下两种：
1、为实现向您提供我们产品及/或服务的基本功能，您须授权我们收集、使用的必要的信息。如您拒绝提供相应信息，您将无法正常使用我们的产品及/或服务；
2、为实现向您提供我们产品及/或服务的附加功能，您可选择单独同意或不同意我们收集、使用的信息。如您拒绝提供，您将无法正常使用相关附加功能或无法达到我们拟达到的功能效果，不会影响您使用我们的基本功能。
我们在此特别提醒您：我们致力于打造多样的产品和服务以满足您的需求。因我们向您提供的产品和服务种类众多，且不同用户选择使用的具体产品/服务范围存在差异，故基本/附加功能及对应收集使用的个人信息类型、范围会有所区别，请以具体的产品/服务功能为准。

（一）账号注册、身份认证与基础咨询中个人信息的收集使用
1、在您使用方寸悦动产品或服务过程中，我们将基于以下需要收集、使用、共享、存储您的个人信息：
（1）账号注册：当您使用方寸悦动产品或服务时，需要您提供手机号码，完成短信验证后，注册成为方寸悦动用户。
（2）咨询师身份认证：在咨询师发布咨询服务之前，需通过实名认证与实人认证。认证过程中，我们需核验您的身份信息（头像、姓名、性别、地区、从业年限、擅长领域、教育背景、资格证书、详细介绍、证件类型、身份证号、身份证件图片等）和执业信息。
（3）个人预约咨询：为匹配可能合适咨询师及心理服务，提高匹配精准度及咨询满意度，咨询用户在注册成功后，需要填写个人信息（头像、姓名、性别、行业等）以完善在线个人信息；在咨询前，您需要先填写咨询信息（头像、姓名、性别、行业、问题类型、问题描述、期望效果等）。
（3）位置有关的服务：为更好地向您提供咨询功能以及与位置有关的产品或服务，在查看或推荐附近咨询师、填写个人地址、用户身份认证、发布或修改咨询信息、与用户聊天等场景，我们会收集您的位置信息，您可以按需选择使用或关闭。
（4）在线支付：在您选择在线支付购买我们的方寸币、虚拟道具、付费权益等增值服务时，我们需要将您的购买明细、订单号与交易金额信息与第三方支付机构共享。
（5）在线聊天：当您使用方寸悦动与用户/咨询师进行线上沟通时，可根据自身意向，与对方交换个人信息。当您选择使用在线语音/视频通话功能前，需要开启您的麦克风及/或相机权限。
（6）直播咨询：您可以根据自身需要，选择使用方寸悦动提供的直播咨询服务。如果您是咨询师，在您使用直播咨询服务前，需要开启您的相机及麦克风权限。请您知悉，如您拒绝授权，我们将无法为您提供相应的服务，但不影响您使用方寸悦动的其他产品或服务。
（7）评论功能：您可通过评论功能公开发布信息，相关页面将显示您的用户名。您可以按需针对头像、昵称信息进行相关隐藏设置。
（8）个性化推荐服务：我们会基于您填写的行年龄、性别、行业、个人困扰选择情况等向或咨询意向以及您的操作记录，为您推荐、展示您可能感兴趣的或与您较为匹配的课程、测评、咨询或平台其他产品或服务，以提升您的心理服务满意度及使用体验，您可以选择关闭该服务。
（9）咨询安全保障：我们会在特定场景下收集您的通讯信息、身份信息、位置信息、操作行为、日志及设备信息等，以保障您的账号安全及交易安全。
对于上述您的个人信息，大部分是由您主动向我们提供的，也有我们在您使用我们的产品或服务时通过Cookie、SDK及类似技术获取的。
请您理解，我们无法单独向您提供方寸悦动全部的服务，我们会为实现特定功能和提供特定服务同我们的关联公司及其他服务提供商一起为您提供这些服务。因此，为满足为您提供特定产品或服务的必要，我们需要向我们的关联公司、合作伙伴及其他第三方共享您的部分必要的个人信息。我们承诺在共享过程中，尽最大可能保障您的数据和隐私不受侵害，并以不低于本政策所要求的保密和安全措施来处理该信息。
您可以通过多种方式管理您的个人信息，例如自行访问、修改、删除您的个人信息、改变授权范围，以及注销您的账户。相关处理途径您可以通过以下个人信息保护政策正文详述具体了解。如果您在管理您的个人信息时遇到问题，您还可以通过下方联系方式与我们联系。在此过程中，如果您有任何疑问、意见或建议的，我们会尽快为您作出解答。但也请您理解，我们可能会因为履行法律法规义务或者我们提供服务必要性的原因无法完全满足您的请求。
（三）需要您授权同意调取系统权限的情形
1.基于保护咨询过程中的咨询者人身安全之考虑，以及满足互联网平台服务普遍存在的风控需求，方寸悦动的附加业务功能可能需要您事先同意我们调取相关的系统权限以收集和使用您的个人信息。您可通过方寸悦动APP上的【我的】-【设置】-【账号与安全】-【权限管理】逐项查看具体的个人信息访问权限及其开启状态，并决定开启或关闭这些权限。若开启这些权限即代表您授权我们收集和使用这些个人信息。我们仅会在您使用相关功能时使用您的系统权限。
2、关闭上述系统权限，即代表您撤回这些授权，我们将不再继续收集和使用您的这些个人信息，但也无法为您提供与这些授权所对应的相关功能。上述权限的关闭决定并不会影响此前基于您的授权所进行的个人信息的处理。

（四二）向您展示商品或服务信息
为向您展示更契合您需求的商品或服务信息，我们会收集和使用您在访问或使用方寸悦动平台或移动客户端时的浏览、搜索记录及设备信息、服务日志信息，以及其他取得您授权的信息，通过算法模型预测您的偏好特征，匹配您可能感兴趣的商品、服务或其他信息，并根据您点击、浏览或购买的情况，对向您展示的商品、服务或其他信息进行排序。为满足您的多元需求，我们会在排序过程中引入多样化推荐技术，拓展推荐的内容，避免同类型内容过度集中。我们也会基于您的偏好特征在方寸悦动平台向您推送您可能感兴趣的商业广告，或者向您发送商业性短信息。
其中：设备信息是您在访问及/或使用我们服务过程中，我们接收并记录您所使用的设备相关信息（包括设备型号、操作系统版本、设备设置、IP地址、设备标识符、设备环境、移动应用列表等软硬件特征信息）、设备所在位置相关信息（包括您授权的GPS位置以及WLAN接入点、蓝牙和基站等传感器信息）。
服务日志信息是您在访问及/或使用我们服务过程中，我们自动收集和记录您的详细使用情况，包括浏览记录、点击查看记录、搜索查询记录、收藏、交易、售后、关注分享信息、发布信息，以及IP地址、电信运营商、使用语言、访问日期和时间。
同时，我们通过以下措施努力保障您的隐私体验：
1、如您不想接受我们给您发送的商业广告，您可通过短信提示回复退订或我们提供的其他方式进行退订或关闭。

（五三）为您提供收藏、关注功能
在您浏览我们平台的过程中，您可以选择对感兴趣的商品及/或服务进行收藏、与您感兴趣的咨询师建立关注关系。在您使用上述功能的过程中，我们会收集包括您的收藏及关注关系的历史在内的服务日志信息用于实现上述功能及其他我们明确告知的目的。您可以在收藏、关注中管理或删除您的收藏和关注信息。
（六四）帮助您完成下单及订单管理
当您在我们的产品及/或服务中订购具体商品及/或服务时，我们会通过系统为您生成购买该商品及/或服务的订单。为完成预约咨询，在下单过程中，您需至少提供您的姓名、性别、年龄、行业或咨询问题描述等个人信息，对于部分特殊类型的商品和服务您还需要提供该商品或服务所必需的其他信息。
您提供的上述信息，仅限您申请的咨询师可查看，且该咨询师有责任进行保密；如有法律纠纷等特殊情况，可能会被依法使用，以保障您的合法权益。
您提供的上述信息，将在您使用本服务期间持续授权我们使用。
同时该订单中会载明您所购买的商品及/或服务信息、具体订单号、订单创建时间、您应支付的金额、您的订单信息。我们收集这些信息是为了帮助您顺利完成交易、保障您的交易安全、便于您查询订单信息、提供客服与售后服务及其他我们明确告知的目的。
为便于您了解、查询订单信息并对订单信息进行管理，我们会收集您在使用我们服务过程中产生的订单信息用于向您展示及便于您对订单进行管理。 您可以通过“我的订单”查看和管理您的订单信息。
（七五）帮助您完成支付
在线支付：在您选择在线支付购买我们的方寸币、虚拟道具、付费权益等增值服务时，我们需要将您的购买明细、订单号、与交易金额信息及其他反洗钱法律要求的必要信息与第三方支付机构共享。为使我们及时获悉并确认您的支付进度及状态，为您提供售后与争议解决服务，您同意我们可自第三方支付机构处收集与支付进度相关的信息。
（八六）帮助向您完成商品或服务的交付
为向您提供售后与争议解决服务之目的，我们需要及时获悉并确认交付进度及状态，根据您与商品及/或服务提供主体的约定，相关配送服务主体（包括商家或第三方配送服务提供商）将为您完成订单的交付。您知悉并同意相应配送服务主体不可避免地获知及使用您的订单相关配送信息（如：收货人姓名、收货地址、收货人联系电话），以保证您购买/申请的商品及/或服务能够顺利、安全、准确送达。您同意我们可自交易相关服务处收集与交付进度相关的信息。
（九七）客服及争议处理
当您与我们联系或提出售中、售后、争议纠纷处理申请时，为了保障您的账户及系统安全，我们需要您提供必要的个人信息以核验您的身份。
为便于与您联系、尽快帮助您解决问题或记录相关问题的处理方案及结果，我们可能会保存您与我们的沟通、通信/通话记录及相关内容（包括账户信息、订单信息、您为了证明相关事实提供的其他信息，或您留下的联系方式信息），如果您针对具体订单进行咨询、投诉或提供建议的，我们会使用您的账户信息和订单信息。
为了提供服务及改进服务质量的合理需要，我们还可能使用的您的其他信息，包括您与客服联系时您提供的相关信息，您参与问卷调查时向我们发送的问卷答复信息。
