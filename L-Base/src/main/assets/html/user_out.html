1. 1、与您约定免除或限制责任的条款
2. 2、其他以粗体标识的重要条款

当您按照注销页面提示填写信息、阅读并同意本《注销协议》及相关条款与条件且完成全部注销程序后，即表示您已充分阅读、理解并接受本《注销协议》的全部内容。阅读本《注销协议》的过程中，如果您不同意相关任何条款和条件约定，请您立即停止账户注销程序。我们在此善意地提醒您，您注销账户的行为会给您的售后维权带来诸多不便，且注销方寸账户后，您的个人信息我们会在方寸的前台系统中去除。请确保您所申请注销的方寸帐号应当是您本人依照《方寸悦动用户服务协议》的约定创建并由方寸悦动提供给您本人的帐号。
1. 1、如果您仍执意注销账户，您的账户需同时满足以下条件:
(1) (1)自愿放弃账户在方寸系统中的资产和虚拟权益(包括但不限于账户余额、VIP卡余额、方寸币、优惠券、会员账户权益等)，并确保账户中无欠款;
(2) (2)账户内无未完成的订单和服务；
(3) (3)账户无正在进行中的圾诉举圾或被投诉举报；
(4) (5)账户对应的方寸支付账户已注销，且已妥善处理了方寸、支付账户下的所有问题；
(5) (6)账户已经解除与其他网站、其他APP的授权登录或绑定关系；
(6) (7)账户当前为有效账户，非冻结状态；
(7) (8)实名账户已解除实名信息。
(8) 在您申请注销帐号后，方寸悦动会在完成合理和必要的验证后协助您注销帐号，删除有关您帐号的一切信息，法律法规另有规定的除外。
2. 2、方寸账户一旦被注销将不可恢复（即使您使用相同的手机号码再次注册并使用方寸悦动），请您在操作之前自行备份方寸账户相关的所有信息和数据。请您保存好订单商品和服务的交易凭证、票据等资料，否则您有可能须支付额外的账户和订单查询费用，或无法享受售后服务。
3. 3、如果您的方寸账户同时是绑定企业账户，您需先解除相关绑定。
4. 4、在方寸账户注销期间，如果您的方寸账户涉及争议纠纷，包括但不限于投诉、举报、诉讼、仲裁、国家有权机关调查等，方寸有权自行终止本方寸账户的注销而无需另行得到您的同意。
5. 5、注销方寸账户，您将无法再使用本方寸账户，也将无法找回您方寸账户中及与账户相关的任何内容或信息，包括但不限于：
(1) (1)您将无法登录、使用本方寸账户；
(2) (2)方寸账户的个人资料和历史信息(包括但不限于用户名、头像、购物记录、关注信息等)都将无法找回；
(3) (3)您通过方寸账户使用、授权登录或方寸第三方的其他服务的所有记录将无法找回。您将无法再登录、使用前述服务，您曾获得的余额、优惠券、积分、资格、订单、VIP账户权益及其他卡券等视为您自行放弃，将无法继续使用。您理解并同意，方寸无法协助您重新恢复前述服务。请您在提交注销申请前，务必先了解您须解绑的其他相关账户信息，具体可与我们的客服联系。
6. 方寸账户销将导致方寸悦动终止为您提供服务，依《方寸悦动用户服务协议》和《隐私政策》等约定的双方权利义务终止，依 法律法规另有规定、本页面其他条款另行约定不得终止的或依其性质不能终止的除外。
6.7. 6、注销本方寸账户并不代表本方寸账户注销前的账户行为和相关责任得到豁免或减轻。
